## 問題根因彙整

| 問題 | 根本原因 | 解法 |
|------|----------|------|
| Docker Engine 無法啟動 | WSL 為舊版 inbox（WSL 1），不支援 Docker backend | `wsl --install --web-download` → WSL 2.6.3.0 |
| pnpm 無法辨識 | Node.js 已安裝但未全域安裝 pnpm | `npm install -g pnpm` |
| npx 執行被封鎖 | PowerShell 執行政策預設 Restricted | `Set-ExecutionPolicy RemoteSigned` |
| `docker ps \| grep` 語法錯誤 | grep 為 Unix 指令，PowerShell 不支援 | 改用 `Select-String` 或 `--filter` 參數 |
