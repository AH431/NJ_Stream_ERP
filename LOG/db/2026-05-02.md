# DB Migration Log - 2026-05-02

This file is append-only. `scripts\migrate_with_log.ps1` records guarded migration runs here.

## Policy

- Before migration: create encrypted PostgreSQL backup unless `-SkipBackup` is used.
- Record data movement: pre/post row counts for every public table.
- Record API change context: uncommitted backend route/app changes from git.
- Record migration context: tracked Drizzle SQL migration files.
- Automatic rollback: if migration fails or row counts drop for existing tables, restore the pre-migration backup with `restore_pg.ps1 -DropAndRecreate -Force`.
- Intentional destructive migrations must be run with `-AllowRowCountDrop` and documented in this log.
