## K-01 ｜ MCU 微控制器

### 2026 供應鏈現況

- 去庫存化於 2025 Q3 基本結束，2026 H1 進入結構性缺貨階段
- 主因：AI 邊緣運算設備（Edge AI）與低軌衛星地面端需求暴增，消耗大量工業級 40nm MCU 產能
- STM32F 系列標準品交期回復至 16–20 週，但特殊溫度規格（-40°C ~ 105°C）仍達 40 週以上
- 中國市場因出口管制壓力，加速國產替代（GD32、CH32、W806）

### 市場趨勢

- **RISC-V 崛起**：CH32V 系列（WCH）在成本敏感設計中佔比從 8% 升至 22%（2025→2026 估計）
- **晶片安全認證**：PSA Certified Level 2 成為歐盟 CRA 法規要求，採購 MCU 需確認認證狀態
- **WiFi + MCU SoC**：ESP32-S3 / ESP32-C6 整合度高，減少 BOM 數量，但 EoL 風險需納入評估

### 採購實務建議

1. **雙源認證（Dual Source Qualification）**：每款 MCU 同步完成一款 Pin-to-Pin 替代料認證，優先選 GD32 或 APM32
2. **韌體抽象層（HAL Layer）**：撰寫硬體抽象層，讓韌體能在 STM32 / GD32 之間無縫切換，改板成本降至 1–2 天
3. **交期監控**：每週查詢 Mouser / Digi-Key 到貨日，超過 26 週即啟動替代料評估

### AI 助理備選問題

```
Q: STM32F103C8 缺貨，有什麼替代品？
A: Pin-to-Pin 相容方案：GD32F103（兆易創新）、APM32F103（極海半導體）、CH32F103（WCH）。
   建議先確認韌體是否使用 STM HAL Library，若是，移植至 GD32 約需 1–2 天。
   2026 H1 GD32F103 LCSC 現貨充足，交期 1 週以內。

Q: ESP32-WROOM32U 交期多久？
A: 2026 Q2 Mouser/Digi-Key 庫存正常，交期約 4–8 週。
   若需立即交貨可查 LCSC 現貨（深圳庫），通常 1–3 個工作天。
   替代方案：W806（聯盛德），WiFi+BT 整合，成本較低但 SDK 生態系較小。

Q: MCU 如何防止被單一供應商鎖定？
A: 採用「功能等效替代清單（Approved Vendor List, AVL）」設計策略：
   設計初期即列入 2–3 個供應商，並完成至少一款替代品的韌體驗證。
```

## K-02 ｜ PMIC 電源管理 IC

### 2026 供應鏈現況

- TI 主流 PMIC（BQ24xxx、TPS6xxx 系列）交期 12–20 週，高效率同步整流型達 28 週
- Analog Devices (ADI) 收購 Maxim 後整合陣痛，部分型號延遲至 2026 Q3 才恢復正常交期
- 中國本土替代（SY/ETA 系列）品質參差，建議僅用於消費性產品，工業級慎用

### 市場趨勢

- **GaN 電源 IC**：Navitas / GaN Systems 的氮化鎵元件開始進入中功率（10–100W）市場，轉換效率提升 5–8%
- **USB PD 3.1 EPR**：高達 240W 規格推動 PMIC 替換需求，Cypress（Infineon）CYPD 系列需求大增
- **功耗法規**：EU ErP 指令 2026 更新版要求待機功耗 ≤ 0.5W，驅動設計端選用更低靜態電流 PMIC

### 採購實務建議

1. 設計時選用「工業級溫度範圍 + 商業等級價格」的 PMIC，避免量產被迫升級到工業料單價
2. 預留 PCB 兩種 PMIC 封裝 footprint（如 SOT-23-5 + DFN-8），便於緊急換料
3. 小批量採購優先使用 Digi-Key 現貨，50 顆以上走代理商取得更好的價格與交期保障

### AI 助理備選問題

```
Q: TI TPS62840 缺貨，替代方案？
A: 功能相容替代：SY8089（Silergy）、MT3540（Microtek）、ETA3425（ETA Solutions）。
   注意：靜態電流與軟啟動時間需重新驗證，建議在 EVB 上確認輸出紋波後再定板。

Q: USB PD 控制器有哪些選擇？
A: 主流方案：
   - Cypress CYPD3177（Infineon）：支援 PD 3.0，適合 65W 以下
   - IP2368（英集芯）：PD 3.1 EPR 支援，CP 性價比高，國內供貨穩定
   - FUSB302（onsemi）：開源 PD Stack，社群支援豐富
```

## K-03 ｜ MLCC 積層陶瓷電容

### 2026 供應鏈現況

- 村田（Murata）與 TDK 針對高容值（10µF+ / 0402）實施**季度配額制（Allocation）**
- 主要壓力來源：電動車（EV）每台用量達 15,000 顆 MLCC，5G 基站持續擴建
- 標準品（1µF/0.1µF 0402）目前供應正常，高溫（X7R 125°C）與高壓（50V+）仍緊缺

### 市場趨勢

- **國產替代成熟**：風華高科、三環集團品質已達日系 85–90% 水準，適用於消費與工業低壓場合
- **尺寸小型化**：0201 尺寸需求年增 35%，對應 AI 可穿戴設備與 TWS 耳機市場
- **價格走勢**：2026 Q1 標準品回落 10–15%，但高壓 / 高溫規格持續漲價

### 採購實務建議

1. **分級庫存策略**：
   - 標準品（0402/0603, X5R/X7R, 1µF, 10µF）：維持 **3 個月安全庫存**
   - 特殊規格（高溫 / 高壓）：維持 **6 個月安全庫存**
2. BOM 中每款 MLCC 標注「可接受等效替代廠牌」，避免驗收時被綁定單一原廠
3. 與代理商簽訂「貨量預留協議（Blanket Order）」，鎖定未來 6 個月出貨量

### AI 助理備選問題

```
Q: 村田 GRM188R71C105KA12 缺貨，替代品？
A: 規格：0603, 1µF, X7R, 16V
   替代選項：
   - TDK C1608X7R1C105K（尺寸/規格完全對應）
   - Samsung CL10B105KA8NNNC（韓系，性價比優）
   - 風華 0603B105K160NT（國產，價格低 30%，消費級適用）
   驗證重點：容值溫度係數 (TCC) 與 ESR 需與原設計確認。

Q: 電容採購如何避免漲價風險？
A: 建議三步驟：
   1. 與代理商簽 Blanket Order（預購合約），鎖定半年價格
   2. 為高風險規格建立雙廠認證 (AVL)
   3. 使用 BOM 健康度工具（如 Octopart API）每月掃描庫存水位
```

## K-04 ｜ 電阻與被動元件

### 2026 供應鏈現況

- 標準厚膜電阻（0402/0603 1% 精度）供應充足，交期 4–8 週
- 精密電阻（0.1% 以下）與高功率型（1W+）仍維持 12–16 週
- 主要供應商：Panasonic、KOA、Yageo；國產品牌（ACR、EVER）適合非關鍵場合

### 市場趨勢

- **環保合規壓力**：RoHS 3.0 草案收緊鉛錫限制，2026 H2 可能影響部分高溫錫膏兼容型號
- **功率密度需求上升**：小尺寸大功率電阻需求增長，推動 0402 2W 型號出現

### 採購實務建議

1. 一般設計中 0402/0603 電阻不設定廠牌，標注「Yageo 或同等品」即可降低缺貨風險
2. 精密電阻需提前認證替代廠牌，避免單一 Vishay / Susumu 供貨中斷影響交期

### AI 助理備選問題

```
Q: 薄膜電阻和厚膜電阻有什麼差別？採購哪種？
A: 厚膜電阻：精度 1–5%，成本低，適合一般電路（推薦日常採購）
   薄膜電阻：精度 0.01–0.1%，溫度係數小，適合量測電路、ADC 前端
   建議：除非設計有精度需求，否則一律採用厚膜以降低成本與交期風險。
```

## K-05 ｜ 鋰電池與電芯

### 2026 供應鏈現況

- 碳酸鋰現貨價格從 2023 年高峰下跌 75%，目前維持在 $10–12 USD/kg 區間
- 中國電芯產能過剩，18650 / 21700 圓柱電池現貨充足，交期 2–4 週
- **地緣政治風險**：美國 IRA 電池來源認證規定影響出口北美市場的產品，需確認電芯產地

### 市場趨勢

- **磷酸鐵鋰（LFP）普及**：循環壽命長達 3,000 次，安全性優，正快速替代三元電池（NMC）用於工業儲能場合
- **固態電池商業化延遲**：Toyota / 三星固態電池量產時程延至 2027–2028，2026 年採購仍以液態鋰電為主
- **電池護照法規（EU Battery Passport）**：2026 年起歐盟要求附帶數位電池護照，影響 OEM 追溯體系

### 採購實務建議

1. 設計 PCB 時預留多種封裝空間（如 18650 與聚合物電池），增加供應商彈性
2. 出口歐美市場產品，提前申請 UL2054 / IEC62133 認證，避免換料時重新認證
3. 量產前完成 2–3 家電芯替代認證，降低單一供應商風險

### AI 助理備選問題

```
Q: 電池缺貨時如何快速換料？
A: 事前準備是關鍵：
   1. 設計時預留不同電芯封裝的 footprint 與固定孔位
   2. BMS 設計需支援充電曲線參數可設定（CC/CV 電流可調）
   3. 提前完成 2–3 顆替代電芯的充放電循環與安全測試
   4. 檢查換料是否影響現有認證（CE/UL）— 若有需提前申請補充認證

Q: LFP 和 NMC 電池怎麼選？
A: LFP（磷酸鐵鋰）：安全性高，循環壽命 2,000–3,000 次，能量密度低，適合工業/儲能
   NMC（三元鋰）：能量密度高，適合需要輕薄設計的消費性產品，但熱失控風險較高
   2026 建議：工業設備優先 LFP，可穿戴 / 手持設備選 NMC
```

## K-06 ｜ 能源管理模組

### 2026 供應鏈現況

- TP4056 類鋰電充電 IC 供應過剩，現貨充足且價格持續下探（$0.05 USD 以下）
- 高整合度 PMIC（充電 + 昇壓 + 保護三合一）交期 8–14 週，國產 IP5xxx 系列貨源穩定

### 採購實務建議

1. 簡單單電池應用可使用 TP4056（成本最低）
2. 需升壓功能的應用選 IP5306 / IP5328（英集芯），整合度高節省 BOM 複雜度
3. 高端設計選 TI BQ25895 / BQ40Z50，支援 I2C 讀取電量與健康狀態

### AI 助理備選問題

```
Q: TP4056 有哪些升級替代方案？
A: 依功能需求分級：
   - 基礎充電（僅充電保護）：TP4056（現貨豐富，成本最低）
   - 充電 + 昇壓（5V 輸出）：IP5306（英集芯），單晶片解決方案
   - 充電 + 電量計 + I2C 通訊：TI BQ25895M（適合需要精確電量顯示的產品）
```

## K-07 ｜ WiFi / BT 通訊模組

### 2026 供應鏈現況

- ESP32 系列（Espressif）供應穩定，LCSC 常備庫存，交期 1–3 週
- WiFi 4/5 獨立驅動 IC（如 Marvell 88W8XXX 系列）陸續宣告 EOL（End of Life）
- WiFi 7（802.11be）模組開始量產，但成本仍為 WiFi 6 的 2–3 倍

### 市場趨勢

- **Thread / Matter 協定**：智慧家居標準整合加速，新設計優先考慮 Matter 相容晶片（如 ESP32-H2）
- **UWB 超寬帶**：Apple/Samsung 生態系帶動 UWB 定位需求，DW3000（Qorvo）採購量增加
- **模組認證**：FCC / CE 預認證模組（如 ESP32-WROOM-32E）可縮短產品上市時程 2–4 個月

### 採購實務建議

1. **優先採用模組（Module）而非裸晶（Bare Die）**，以節省天線設計與 RF 認證成本
2. 確認所選模組的 Wi-Fi 標準與目標市場規格一致（如日本需要 TELEC 認證）
3. 針對 EOL 零件，查詢官方「Last Time Buy（LTB）」截止日期，及時補庫存

### AI 助理備選問題

```
Q: ESP32 缺貨，有什麼替代方案？
A: 依應用場景選擇：
   - WiFi + BT（低功耗優先）：W800/W806（聯盛德 WinnerMicro），國產，LCSC 現貨
   - WiFi + BT（Arduino 生態）：Nordic nRF7002（僅 WiFi companion IC）或 Realtek RTL8720DN
   - Matter / Thread 支援：ESP32-H2（官方持續供貨）、SiLabs MGM240S
   注意：替換時需評估 SDK 差異，W800 SDK 生態系比 ESP-IDF 小，移植成本較高。

Q: 舊產品用的 WiFi IC 停產了怎麼辦？
A: 三步驟處理：
   1. 查官方停產公告 (PCN/EOL Notice)，確認 Last Time Buy 截止日
   2. 評估「模組化替代」— 用 ESP32-WROOM 模組替換獨立 WiFi IC + MCU 架構
   3. 如無法換模組，聯繫現貨商採購 EOL 零件備料，並啟動下一代產品改板計畫
```

## K-08 ｜ 顯示模組

### 2026 供應鏈現況

- 小尺寸 OLED（0.96"–2.4"）供應充足，中國廠商（中景園、微雪）主導低階市場
- TFT LCD 驅動 IC（ILI9341 / ST7789）庫存充裕，價格持平
- 大尺寸（5" 以上）工業 LCD 交期 6–10 週，面板廠產能移往 TV / 車用市場

### 市場趨勢

- **AMOLED 下沉**：2.4" 以下 AMOLED 成本接近 TFT，開始進入工業 HMI 市場
- **低功耗 E-Ink 普及**：IoT 電池設備大量採用電子紙（E-Ink）顯示，Good Display / GDEW 系列需求上升

### AI 助理備選問題

```
Q: SSD1306 OLED 模組有更省電的替代品嗎？
A: 是的，依功耗需求分級：
   - 超低功耗靜態顯示：電子紙（E-Ink），如 GDEW0154M10，刷新才耗電
   - 動態 + 省電：SH1107（方形 OLED），靜態電流 <0.5mA
   - 彩色低功耗：GC9A01（圓形 TFT），有 partial refresh 功能
   
Q: 顯示模組如何做到模組化設計？
A: 推薦「子板（Daughter Board）」架構：
   - 主板保留 SPI/I2C 通用接口 + 標準 FPC 連接器
   - 顯示模組獨立成子板，面板缺貨時只換子板，主板不改版
   - 開發初期預先驗證 2 款不同廠牌顯示模組，確認接口相容性
```

## K-09 ｜ 採購通用策略

### 2026 市場總體趨勢

| 趨勢 | 影響 | 建議對應 |
| :--- | :--- | :--- |
| AI 邊緣運算爆發 | MCU / NPU 結構性缺貨 | 提前 AVL 認證替代 MCU，建立 6 個月備料 |
| 地緣政治分裂 | 中美雙鏈供應需求 | BOM 中標注美系 / 中系替代方案，雙線認證 |
| 歐盟 CRA / RoHS 收緊 | 合規認證成本上升 | 新設計階段即確認零件合規狀態 |
| 原物料價格回穩 | MLCC / 被動元件成本下降 | 鎖定低價窗口簽訂 Blanket Order |
| WiFi 7 / Matter 普及 | 舊通訊 IC 加速 EOL | 每季審查 BOM 中通訊元件的停產風險 |

### 採購通用應變策略

| 處置方式 | 說明 | 適用時機 |
| :--- | :--- | :--- |
| **混合採購** | 同時向一級代理（低價長交期）與現貨商（高價即時）下單 | 緊急缺料，需量小於 500 顆 |
| **BOM 健康檢查** | 每月用 Octopart / Z2Data 掃描 BOM，識別 EOL / Single Source 風險 | 例行維護，每月一次 |
| **Blanket Order** | 向代理商預購未來 6 個月用量，鎖定價格與交期 | 年度採購規劃，Q4 執行 |
| **替代料改板** | 交期 > 52 週時啟動換料設計，而非被動等待 | 長期缺貨且無現貨時 |
| **AVL 雙廠認證** | 每款關鍵零件維護 2–3 個通過驗證的替代廠牌清單 | 新品設計 EVT 階段完成 |

### AI 助理備選問題

```
Q: 如何快速評估 BOM 的缺料風險？
A: 三步驟快速評估：
   1. 匯出 BOM 至 Octopart 或 Findchips，批量查詢庫存水位與交期
   2. 標記風險等級：庫存 < 1,000 顆 或 交期 > 26 週 → 高風險
   3. 高風險料號立即啟動：確認現貨商報價 + 找替代料 + 評估改板可行性

Q: 現貨商和代理商的採購有什麼差別？
A: 代理商（Authorized Distributor）：原廠授權，保證品質與來源，適合量產長期採購
   現貨商（Spot / Broker）：價格較高，可立即出貨，風險在於可能有仿品
   建議：現貨商採購前要求原廠 COC（Certificate of Conformance）和 Date Code，
   並考慮送樣至第三方進行 IC 真偽檢測（如 SGS、台灣工研院）

Q: 如何建立採購預警機制？
A: 推薦以下機制：
   1. 每月自動跑 BOM vs 庫存比對（可整合 Octopart API）
   2. 設定「安全庫存警戒線」，低於警戒線自動產生採購建議單
   3. 訂閱原廠 PCN（Product Change Notification）郵件
   4. 將前述資訊整合至 ERP 系統，由 AI 助理定期產出風險摘要報告
```
